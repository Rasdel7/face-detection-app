# Face Detection App 👤

Detects human faces in images using
OpenCV Haar Cascade classifier.

## Live Demo
[Click here](https://face-detection-app-kmd7ors95xdufmdpqpw6us.streamlit.app)

## Features
- Upload any image for face detection
- Adjustable detection parameters
- Bounding box color customization
- Face coordinates and area table
- Real-time detection results

## How It Works
Uses OpenCV's pre-trained Haar Cascade
classifier — the same algorithm used in
digital cameras worldwide.

## Tools Used
- Python, OpenCV, Streamlit, Pillow, NumPy

## How to Run Locally
pip install streamlit opencv-python-headless Pillow numpy pandas
streamlit run app.py
